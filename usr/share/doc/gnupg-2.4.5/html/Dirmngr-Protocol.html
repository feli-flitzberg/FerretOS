3.6 Dirmngr’s Assuan Protocol ¶

Assuan is the IPC protocol used to access dirmngr. This is a description of the commands implemented by dirmngr.

  • Return the certificate(s) found
  • Validate a certificate using a CRL or OCSP
  • Validate a certificate using a CRL
  • Validate a certificate using OCSP
  • Put a certificate into the internal cache
  • Validate a certificate for debugging
