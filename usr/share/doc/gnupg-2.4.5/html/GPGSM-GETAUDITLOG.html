5.6.10 Retrieve an audit log ¶

This command is used to retrieve an audit log. 

GETAUDITLOG [--data] [--html]

If --data is used, the audit log is send using D-lines instead of being sent to the file descriptor given by an OUTPUT command. If --html is used, the output is formatted as an XHTML block. This is designed to be incorporated into a HTML document.
